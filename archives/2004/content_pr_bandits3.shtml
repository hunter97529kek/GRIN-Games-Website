"BANDITS - Phoenix Rising" 1.1.1 patch out now!

GRIN now posts the "BANDITS - Phoenix Rising" patch for all territories

If you have already installed the previous US patch 1.1.1 this is nothing new, only now it works on European and Australian versions as well.

Rightclick here and choose "Save Target As" to download the patch

To install, simply run and extract the downloaded file into the folder on your harddrive in which you have already installed your copy of "BANDITS - Phoenix Rising".
If you have the TriSynergy version of BANDITS, make sure you type in this folder path correctly when prompted.

Patch 1.1.1 modifications: (Note: Some of these fixes are already implemented in the Trisynery's version of the game (the American version))

Modified gamebalance:
  • The game starts with more weapons available, namely the Sharkbite and the Scrapmaker
  • Weapons in general are now recieved earlier in the game
  • All player vehicles have increased armour levels
  • Vincent has decreased armour level
  • Sebastion has decreased armour level
  • The train in mission 13 has increased armour level
  • All enemies have been additionally balanced
Major bugs fixed:
  • Fixed a bug in Windows XP which sometimes made the game lag and then die
Minor bugs fixed:
  • Fixed a bug in mission 3 which could result in a checkpoint not being recognized
  • Fixed a bug in mission 5 that sometimes made the enemies disappear or hard to find
  • Fixed a bug in mission 12 which could result in certain mission information not being presented to the Player
  • Additional fixes to increase stability on certain ATI and nVidia cards, though always make sure you use the latest drivers from ATI and nVidia
Additional candy provided:
  • Added a new multiplayer level
  • All files are now open for moding, such as weapons and effects (just open and edit the xml files to mod)
